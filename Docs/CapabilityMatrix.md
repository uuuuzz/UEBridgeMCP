# UEBridgeMCP Capability Matrix

Source of truth:

- The base editor registration source is `Source/UEBridgeMCPEditor/Private/UEBridgeMCPEditor.cpp :: RegisterBuiltInTools()`.
- The authoritative runtime inventory is `tools/list`.
- The authoritative runtime count is `initialize.capabilities.tools.registeredCount`.
- After Step 6, tool count is intentionally dynamic because conditional tools and extension-module tools may appear or disappear.

Status legend:

- `Existing`: stable first-class surface exists today.
- `Partial`: the capability exists, but v1 limits or coverage gaps remain.
- `Missing`: no dedicated surface yet.
- `Not in core`: intentionally isolated from the core editor module and only exposed through an optional extension module.

As of `2026-04-24`:

- Base editor surface: expanded with the 2026-04-24 UnrealMCPServer parity batch; use `tools/list` for the exact runtime count.
- Additional runtime tools may be contributed by conditional editor integrations and Step 6 extension modules.
