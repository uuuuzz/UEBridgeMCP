# Sequencer Edit Recipe

1. Resolve or create the target sequence asset and confirm playback range before editing.
2. Create or reuse bindings before adding tracks or sections.
3. Prefer the narrowest track needed for the change: camera cut, transform, float, bool, or animation.
4. Add section ranges before key authoring when the intended timing window is known.
5. Save and re-query the sequence after writes so timing drift and binding mismatches are visible immediately.
